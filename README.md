This is a bot_ai addon for Garry's Mod. The addon allows the user to control ulx bots with a GUI console as well as with console commands.

# Bot AI menu command:
bot_ai_menu

# Bot commands:
bot_follow "Bot Name"
bot_drive "Bot Name"
bot_jump "Bot Name"
bot_view "Bot Name"
bot_freeze "Bot Name"
bot_dance "Bot Name"
bot_muscle "Bot Name"
bot_crazy "Bot Name"
